Qu'est-ce que Jupyter?

Un ensemble cohérent d'outils (protocoles, applications, bibliothèques) dont le but est d'améliorer le workflow des ingénieurs et scientifiques:

  • depuis la phase exploratoire de leur travail

  • jusqu'à la communication de leurs résultats
  • incluant toute sles étapes intermédiaires

The Jupyter Notebook

  • Environnement de calcul interactif dans le navigateur

  • Format de document "reproductible":
    • Code
    • Texte narratif (markdown)
    • Equations (LaTeX)
    • Images, visualisation
  • Plus de 50 langages de programmation supportés
  • Open source (licence BSD)

The Jupyter Architecture

Implémentations du protocole

  • Certains kernels fournissent leur propre implémentation du protocole:
    • Python (ipykernel)
    • R (IRkernel)
    • Julia (IJulia)
  • D'autres utilisent le "wrapper kernel" fournit par ipykernel
    • Octave kernel
    • Matlab kernel

"Wrapper kernel"

  • Dépend d'un runtime Python
  • L'interpréteur cible ne fournit pas forcément une API Python
  • Le protocole des widgets n'est pas implémenté
  • Une implémentation native du protocole Jupyter
  • Inclut le protocole des widgets
  • Permet aux développeurs de créer des nouveaux kernels sans avoir à gérer le protocole

Exemple: echo kernel
